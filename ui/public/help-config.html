Help Configuration
  • Model Path : Model path. It must be relative path from model_root
  • Clip Skip : Reduce number of hidden layers of CLIP text encoder. Note that common stable diffusion 1.x uses 12 layers CLIP. Recommend 0, 1 or 2
  • S. Method : Sampling Method. Recommend DPM++ 2M Karras
  • S. Steps : Sampling Steps. Recommend DPM++ 2M Karras
  • CFG Scale, Width, Height
  • Seed : Random seed. Keep it blank if you want to use random seed.
  • Sz.Rng. : Size Range. Must be 0.0-1.0. The width and height will be randomly multiplied by [1.0 - size range, 1.0 + size range]. For example, if width = 500 and Sz.Rng. is 0.5, width will be random size in the range [250, 750]
  • Positive/Negative Prompts Grammer
    • (prompt) increase prompt strength to 1.1
    • ((prompt)) increase prompt strength to 1.21
    • [prompt] decrease prompt strength to 0.9
    • [[prompt]] decrease prompt strength to 0.81
    • (prompt... : 1.5) set prompt strength by 0.81
    • [prompt... : 1.5] set prompt strength by 0.81
    • {a; b; c} Randomly choose one from a, b and c
    • {a; b: 2; c} Randomly choose one from a, b and c, but b will be choosed double probability
    • {a: 2, b: 3;} Randomly choose one from a, b, and empty with different probability
  • Highres Fix Put below into etc 
     {"highres_fix": [{"scale": 1.2, "strength": 0.5}]}
